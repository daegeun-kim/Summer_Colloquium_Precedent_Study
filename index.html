Project Summary (what the author has written about the project):

Mike Bostock:

“I'd say the motivation was to stop reinventing the wheel. The goal of D3 is to embrace and build on standard, universal representations of graphics, rather than reinvent them each time.”

“With D3, the idea is to leverage the native representation … the toolkit isn't a middleman, preventing you from using browser features.”

“By using D3, your code will be shorter and you'll be able to do more. … And better still, you'’'ll get human-readable ticks at multiples of 2, 5 or 10…”

Project Comments & Suggestions (what others have written about the project):

Suggestions
"Use Canvas or WebGL instead of SVG when rendering thousands of elements—greatly improves rendering speed"
"Build interactive UI components (filters, dropdowns, checkboxes) to let users dynamically adjust and explore data views"
"Provide annotation and templating tools so developers can automatically add tooltips, legends, labels consistently across charts"
